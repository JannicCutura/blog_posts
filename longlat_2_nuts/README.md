# CO2 emissions in Europe: Longitude/Latitude -> to NUTS3

### Requirements 
 - python 3.7 and packages in `requirements.txt`
 - NUTS3 shape files (you can download them [here](https://ec.europa.eu/eurostat/de/web/gisco/geodata/reference-data/administrative-units-statistical-units/nuts#nuts21))
 - CO2 emissions data from Emissions Database for Global Atmospheric Research EDGAR (download it [here](https://edgar.jrc.ec.europa.eu/emissions_data_and_maps))
